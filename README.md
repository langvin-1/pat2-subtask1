# Morse Code

## Overview
Morse code is a method of communication that uses dots (.) and dashes (-) to represent letters, numbers, and symbols. It was mainly used for telecommunication systems such as the telegraph.

## History of Morse Code
Morse code was developed in the 1830s and 1840s by Samuel Morse and Alfred Vail. It was created to send messages over long distances using electrical telegraph systems. Morse code became widely used in military communication, aviation, and maritime services.

## How Morse Code Works
Morse code uses combinations of dots and dashes to represent characters.

Examples:
- A = .-
- B = -...
- C = -.-.
- SOS = ... --- ...

A short signal is called a dot, and a longer signal is called a dash. These signals can be sent using sound, light, or electrical pulses.

## Importance of Morse Code
Morse code helped people communicate quickly over long distances before modern communication technologies were developed. Today, it is still used in emergencies and by radio operators.
